Matrices

Identity Matrix, Unit Matrix

It is always a square matrix

An  (square) matrix whose diagonal elements are all 1s and all other elements are 0s.

 

Multiplying any matrix by the unit matrix, gives the matrix itself.

 

An identity matrix is on which when applied to a geometric transformation, the object remains unchanged.

Multiplying Two Matrices

The number of columns in the first matrix must be equal to the number of rows in the second matrix.

 

The product will be a matrix with the dimensions equal to the number of rows of the first term and the number of columns equal to the second term

 

Multiply each row in the first matrix by each column in the second matrix.

 

 

 

Inverse of a Matrix

The inverse of a matrix is another matrix that when multiplied by the given matrix results in the identity or unit matrix.
